  <action>
    In the local /tmp/CalcpadCE-fork clone, on branch `librespin-patched`:

    1. Create `.github/workflows/build-calcpad-cli.yml` with this exact content:

       ```yaml
       name: Build CalcPad CE CLI
       on:
         push:
           tags: ['v*']
         workflow_dispatch:
       jobs:
         build-linux:
           runs-on: ubuntu-latest
           steps:
             - uses: actions/checkout@v4
             - uses: actions/setup-dotnet@v4
               with:
                 dotnet-version: '10.0.x'
             - name: Publish self-contained Cli
               run: |
                 dotnet publish Calcpad.Cli/Calcpad.Cli.csproj \
                   -r linux-x64 \
                   --self-contained true \
                   -p:PublishSingleFile=true \
                   -p:IncludeNativeLibrariesForSelfExtract=true \
                   -c Release \
                   -o ./dist/
             - name: Rename to Cli (assembly name on Linux)
               run: |
                 ls -la ./dist/
                 # Binary is already named 'Cli' per spike finding; this step is a safety check
                 test -f ./dist/Cli || mv ./dist/Calcpad.Cli ./dist/Cli
                 chmod +x ./dist/Cli
             - name: Release
               if: startsWith(github.ref, 'refs/tags/')
               uses: softprops/action-gh-release@v2
               with:
                 files: ./dist/Cli
       ```

    2. Copy this file to `.planning/artifacts/calcpadce-fork-workflow.yml` in the LibreSpin repo: `cp /tmp/CalcpadCE-fork/.github/workflows/build-calcpad-cli.yml /home/william/repo/LibreSpin/LibreSpin/.planning/artifacts/calcpadce-fork-workflow.yml`
    3. Commit + push in the fork clone: `git add .github/workflows/build-calcpad-cli.yml && git commit -m "ci: add linux-x64 self-contained Cli build + release workflow" && git push`
    4. Tag and push: `git tag v0.1.0-librespin && git push origin v0.1.0-librespin`
    5. Wait for CI: `gh run watch --repo $OWNER/CalcpadCE` (or `gh run list --repo $OWNER/CalcpadCE --limit 1`) until status=completed, conclusion=success. If it fails, read logs with `gh run view --log-failed` and fix.
    6. Verify the Release exists and the asset is downloadable:
       `curl -sLI https://github.com/$OWNER/CalcpadCE/releases/latest/download/Cli | head -1` — expect `HTTP/2 200` (after redirect).
    7. Record in 06-01-fork-setup.md:
       - `release_tag: v0.1.0-librespin`
       - `release_url: https://github.com/$OWNER/CalcpadCE/releases/tag/v0.1.0-librespin`
       - `cli_download_url: https://github.com/$OWNER/CalcpadCE/releases/latest/download/Cli`
       - `ci_run_id: <from gh run list>`
       - `verified_at: <ISO date>`
  </action>
