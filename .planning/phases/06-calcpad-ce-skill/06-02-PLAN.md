         Write draft to `/tmp/calcpad-{timestamp}.cpd`.
       - **Stage B: Inline review (D-10/D-11)** — Show the full `.cpd` text in chat. AskUserQuestion: "Approve, edit, or cancel?" If edit, accept corrections in chat, rewrite the file, re-show.
       - **Stage C: Execute CLI (CALC-04)** — If `use_rest: false`:
         ```bash
         "$HOME/.librespin/bin/Cli" "/tmp/calcpad-{ts}.cpd" "/tmp/calcpad-{ts}.html" -s
         EXIT=$?
         if [ $EXIT -eq 0 ] && [ -f "/tmp/calcpad-{ts}.html" ]; then
           echo "OK"
         else
           echo "FAIL exit=$EXIT file=$(test -f /tmp/calcpad-{ts}.html && echo yes || echo no)"
         fi
         ```
         If fail: re-show worksheet, ask user to correct (Pitfall 5 — do not silently retry).
       - **Stage D: REST fallback (CALC-08)** — If `use_rest: true`:
         ```bash
         "$HOME/.librespin/bin/Calcpad.Server" --urls http://localhost:9421 > /tmp/calcpad-server.log 2>&1 &
         SERVER_PID=$!
         sleep 1
         # Verify bound URL from log
         grep "Now listening on:" /tmp/calcpad-server.log || { echo "server failed"; kill $SERVER_PID; exit 1; }
         CPD_JSON=$(python3 -c "import json,sys; print(json.dumps({'Content': open('/tmp/calcpad-{ts}.cpd').read()}))")
         curl -s -X POST http://localhost:9421/api/calcpad/convert \
           -H "Content-Type: application/json" \
           -d "$CPD_JSON" \
           -o "/tmp/calcpad-{ts}.html"
         kill $SERVER_PID
         test -s "/tmp/calcpad-{ts}.html" && echo "OK" || echo "FAIL"
         ```

         ```
         | Target   | Expected | Calculated | Tolerance | Status |
         |----------|----------|------------|-----------|--------|
         | V_out    | 3.3 V    | 3.29 V     | ±2%       | PASS   |
         ```
       - **Stage F: Human review gate (CALC-06)** — AskUserQuestion: "Approve and save, request worksheet changes, or cancel?" Only on "approve" proceed to save.
       - **Stage G: Save outputs (CALC-07)** — `mkdir -p .librespin/08-calculations/` then copy:
         ```bash
         cp /tmp/calcpad-{ts}.cpd .librespin/08-calculations/{block-slug}.cpd
         cp /tmp/calcpad-{ts}.html .librespin/08-calculations/{block-slug}.html
         ```
         Write a `.librespin/08-calculations/{block-slug}-summary.md` with the pass/fail table.
