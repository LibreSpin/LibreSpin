- **cd before running** ‚Äî `wrdata` paths are relative to CWD. Always `cd .librespin/09-simulation/` before running `ngspice -b circuit.cir`. (Pitfall 3)
- **Agent MUST be spawned foreground (`run_in_background: false`)** ‚Äî The simulate agent uses AskUserQuestion for netlist review and result approval. If backgrounded, prompts never surface and the agent burns ~100k tokens before timing out. (Pitfall 6; same as Phase 6 Pitfall 4.)
